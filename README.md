# android-java-lab

Java 기반 Android 학습 레포지토리 모음입니다.  
각 폴더는 **독립적인 예제 Android 프로젝트**로 구성되어 있습니다.

## Projects

### Navigation
- `navigation/StudyNavigation` : Navigation 예제
- `navigation/StudyBottomNavigation` : BottomNavigation 예제
- `navigation/CustomBottomNavigationView` : Custom BottomNavigationView 예제

### Architecture
- `architecture/StudyMVP` : MVP 기반 예제
- `architecture/Study_Room_VM_DataBinding` : MVVM + Room + DataBinding 예제

### Async
- `async/StudyRxJava` : RxJava 학습 예제

### System
- `system/StudyNotification` : Notification 학습 예제

### UI Components
- `ui-components/CommonDialog` : 공통 Dialog 컴포넌트 예제

## How to run
1. Android Studio에서 원하는 프로젝트 폴더를 열기  
   (예: `navigation/StudyNavigation`)
2. Gradle Sync 후 실행

> Note: 이 레포는 “예제 모음” 성격상 한 번에 전체를 빌드하기보다는, 필요한 폴더 단위로 열어 실행하는 것을 권장합니다.
